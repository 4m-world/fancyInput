Fancy Input
=============
Makes typing in input fields fun with CSS3 effects.

##[Demo page](http://dropthebit.com/demos/fancy_input/fancyInput.html)

## Basic use-case example:
    <div class='fancyInput'>
		<input type='text' />
	</div>
    ...
    ...
    ...
    <script>
        $('.fancyInput > input').fancyInput();
    </script>
